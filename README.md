## Project Portfolio: Basic Web Applications

This repository contains three foundational web applications designed to practice core programming logic, API integration, and state management.

---

### 1. Calculator App
A fully functional arithmetic calculator that handles basic mathematical operations. This project focuses on UI layout and input handling.

* **Key Features:**
    * Basic operations: Addition, Subtraction, Multiplication, and Division.
    * Clear (C) and All Clear (AC) functionality.
    * Decimal point support and percentage calculations.
    * Responsive design for mobile and desktop use.
* **Technologies Used:** HTML5, CSS3 (Flexbox/Grid), and Vanilla JavaScript.
* **Logic Implementation:** Uses a listener for button clicks to update the display string and evaluates the mathematical expression using safe parsing methods.

---

### 2. Weather App
A dynamic application that fetches real-time weather data based on user input. This project demonstrates how to work with Asynchronous JavaScript and external APIs.

* **Key Features:**
    * Search by city name.
    * Displays temperature, humidity, and wind speed.
    * Dynamic background or icon changes based on weather conditions (e.g., sunny, rainy, cloudy).
    * Error handling for invalid city names or network issues.
* **Technologies Used:** JavaScript (Fetch API), OpenWeatherMap API (or similar), and CSS for styling.
* **Logic Implementation:** Uses `async/await` to handle API requests and updates the DOM dynamically with the JSON response data.

---

### 3. Quiz App
An interactive multiple-choice quiz that tests knowledge across various topics. This project focuses on managing application state and user progress.

* **Key Features:**
    * Timer-based questions to increase difficulty.
    * Score tracking and a final results screen.
    * Progress bar to show the user's current position in the quiz.
    * Instant feedback on correct/incorrect answers.
* **Technologies Used:** JavaScript (Objects/Arrays), HTML, and CSS.
* **Logic Implementation:** Questions are stored in an array of objects. The app iterates through the array, compares user input to the "correct" key, and increments the score accordingly.

### 4. Tic Tac Toe
A modern Tic-Tac-Toe game with a glowing UI and smart AI.
